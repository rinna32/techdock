# Магазин с товарами

## Описание проекта
Проект предназначен для пользователей, которые бы хотели приобрести различный товар, посмотреть его рейтинг, либо отложить его в избранное для дальнейшей покупки.

## Установка и запуск
Чтобы запустить проект, необходимо открыть терминал, затем перейти в папку проекта с помощью команды:

`cd hhh3`

 после чего ввести команду:

`npm run dev`

 В терминале появится ссылка на localhost проекта, которую нужно скопировать и вставить в браузер, либо нажать сочетание клавиш: на Windows — `Ctrl + ЛКМ`, на MacOS — `Command + ЛКМ`.

## Примеры использования
После запуска вы попадаете на главную страницу, где в верхней части расположено меню. Чтобы перейти к товарам, нажмите кнопку «К товарам». На странице товаров вы увидите карточки товаров, где можно выставить товару рейтинг, добавить его в избранное или в корзину. Также можно посетить страницы «Избранное» и «Корзина».

## Структура репозитория
В папке `hhh3` проекта находится папка `src`, где содержатся корневые файлы: `App.jsx`, в котором подключены основные файлы, а также `routes.js` и `stores.js` — в первом находятся маршруты, во втором контексты для обновления счётчиков избранного и корзины. Внутри `src` также есть папка `components`, где расположены все страницы сайта.

## Технические требования
Для работы проекта необходимы следующие инструменты и технологии: **VSCode**, **React**, **Tailwind CSS**, **react-router-dom**.
