# Магазин с товарами

## Описание проекта
Проект предназначен для пользователей, которые хотят приобрести различные товары, посмотреть их рейтинг или добавить понравившиеся товары в избранное для дальнейшей покупки.



## Установка и запуск

1. Откройте терминал.  
2. Перейдите в папку проекта с помощью команды:
   ```bash
   cd hhh3
   ```
3. Для запуска проекта выполните команду:
   ```bash
   npm run dev
   ```
4. В терминале появится ссылка на локальный сервер проекта (обычно `http://localhost:5173` или похожая).  
   - Чтобы открыть сайт, скопируйте ссылку и вставьте её в браузер.  
   - Или просто зажмите **Ctrl + ЛКМ** (Windows) / **Command + ЛКМ** (MacOS).

---

## Примеры использования

После запуска вы попадаете на **главную страницу** сайта.  
В верхней части расположено меню навигации.

- Чтобы перейти к товарам, нажмите кнопку **«К товарам»** — откроются карточки с товарами.  
- В карточке можно:
  - поставить **рейтинг**;
  - добавить товар в **избранное**;
  - добавить товар в **корзину**.  
- Также доступны отдельные страницы **«Избранное»** и **«Корзина»** для удобного просмотра.

## Структура репозитория

Основная структура проекта расположена в папке `hhh3`:

```
hhh3/
├── src/
│   ├── App.jsx          # Главный компонент приложения
│   ├── routes.js        # Файл с маршрутами
│   ├── stores.js        # Контексты для обновления счетчиков избранного и корзины
│   └── components/      # Компоненты и страницы сайта
└── package.json         # Настройки зависимостей и скриптов
```
